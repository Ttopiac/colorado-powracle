# Palette templates

Color-palette samples we considered for the Colorado Powracle deck. The final
deck (`slides/Colorado_Powracle_AlpineFrost.pptx`) uses **AlpineFrost** — pale
icy blue, inspired by a Colorado bluebird day.

| File | Vibe |
|---|---|
| `Palette_Bluebird_Crisp.pptx`     | bright sky blue, high-contrast |
| `Palette_MountainStone_Dark.pptx` | dark slate, matches the Streamlit Mountain Stone theme |
| `Palette_PowderDay_Light.pptx`    | soft white-on-pale, minimalist |

`build_powder_palettes.py` generates all of the above from a single script.
